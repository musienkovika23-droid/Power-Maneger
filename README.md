# Ultra Power Manager Pro

Це простий та зручний графічний інтерфейс (GUI) для керування живленням вашого комп'ютера на базі Windows. Програма дозволяє планувати вимкнення або перезавантаження системи через заданий проміжок часу, а також швидко скасовувати заплановані завдання.

## 🚀 Основні можливості

- **Вимкнення ПК:** Планування завершення роботи через таймер.
- **Перезавантаження:** Планування перезапуску системи.
- **Скасування:** Миттєва зупинка будь-яких запланованих дій за допомогою команди `shutdown /a`.
- **Візуальний таймер:** Відстеження часу, що залишився до виконання дії, безпосередньо у вікні програми.
- **Інтеграція:** Можливість швидкого переходу до папки з проєктом.

## 🛠 Технології

- **Python 3.x**
- **Tkinter** (стандартна бібліотека для створення GUI)
- **OS** (для системних викликів керування живленням)

## 📋 Вимоги

Для роботи програми вам знадобиться встановлений [Python](https://www.python.org/downloads/). Програма оптимізована для роботи в середовищі **Windows**.

## 📥 Встановлення та запуск

1. Склонуйте репозиторій:
   ```bash
   git clone [https://github.com/ВАШ_НІКНЕЙМ/Ultra-Power-Manager-Pro.git](https://github.com/ВАШ_НІКНЕЙМ/Ultra-Power-Manager-Pro.git)
````

2.  Перейдіть до папки з проєктом:
    ```bash
    cd Ultra-Power-Manager-Pro
    ```
3.  Запустіть скрипт:
    ```bash
    python main.py
    ```

## 💡 Як користуватися

1.  Введіть бажаний час у секундах у текстове поле (за замовчуванням встановлено 60 секунд).
2.  Натисніть кнопку потрібної дії (**Вимкнути** або **Перезавантажити**).
3.  Підтвердіть дію у вікні повідомлення.
4.  Якщо ви передумали, натисніть велику зелену кнопку **СКАСУВАТИ**.

## ⚠️ Важливо

  * Програма використовує системні команди Windows `shutdown`. Переконайтеся, що ви маєте відповідні права доступу для керування живленням системи.
  * Зміни в коді слід робити обережно, оскільки вони впливають на системні процеси.
